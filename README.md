# 🪟Automatic Window Controller

<br/>

###### ● 주제 선정
외출 시 비나 가스 등으로 인해 창문을 열고 닫아야 하는 상황을 해결하기 위한 시스템의 필요성
###### ● 시스템 고안
외부 및 내부 요인에 따라 창문을 자동으로 열고 닫을 수 있는 시스템 고안
###### ● 시스템 구현
랙 앤 피니언 메커니즘을 모터에 적용하고, 수위 센서와 가스 탐지기 등 아두이노 센서 부품을 활용하여 창문 자동화 개폐 시스템을 구현
